# Workout 🚧

A simple timer for circuit training made with ❤️ and elm.

<img width="852" src="https://user-images.githubusercontent.com/17585917/93552487-8c93b380-f93e-11ea-8d91-82f60a33188f.png">

## Features

- Select the number of rounds
- Select each round duration
- Select the rest duration between rounds
- Select a delay before the start of the workout
